Desktop application (Mac OS X)

The desktop application of Aruspix is available for download for Mac OS X (version 10.7 or later is required). If no previous version of Aruspix was installed before on your machine, you need first to download and install the Leipzig font using the Aruspix Font Installer.

Aruspix Font Installer

The latest version of the destkop application of Aruspix can then be downloaded below. Aruspix will eventually automatically check if a new version is available.

Aruspix App

Finally, the following zip archive containing a set of example images and processed files can be dowloaded.

Example Images

Because both the App and the Font Installer are not distributed through the official Apple Store, you need to open then using the "Right-click" and then "Open" from the pop-up menu the first time you run them. See here for more information on how to proceed.

Desktop application

The Aruspix desktop application is currently available for download only on Mac OS X.

Source code

Aruspix source code is available on GitHub. Building instructions for both the desktop application and the command-line version are providing in the repository wiki.
