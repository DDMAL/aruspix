Editing Functions for Clefs

Clefs can only be inserted using the mouse. To change a clef, there are two options:

  • You can select the clef, and then click on the correct clef from the clef family.
  • You can select the clef, and then use the Function Keys to change to clef. THIS FUNCTION IS CURRENTLY DISABLED.
